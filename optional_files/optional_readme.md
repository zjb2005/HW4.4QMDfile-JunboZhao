# Optional Files Directory

This sub-directory (sub-folder) has some additional files that are optional for HW #4.4. If you want to use any of these files, you are free to. You'll need to make sure that you move the files to the appropriate place in your repo/project. Typically, this will be at the same level as your Quarto document.

If you don't use any of these files, you should remove/delete them from your repository. After moving any files, you should also remove/delete this sub-directory.

## Included Files

+ apa7.csl: The citation style language file for using the American Psychological Association's (APA) 7the Edition for references.
+ MLA9.csl: The citation style language file for using the Modern Language Association's (MLA) 9th Edition for references.
+ .lintr: Sets the defaults for linting R scripts, code, and Quarto documents according to the BOAST Style guide.
