# Images Directory

A good organizational practice is to create a dedicated sub-directory (sub-folder) that you place all of your image files in. This can include JPG/JPEGs, PNGs, SVG, GIFs, TIFF, etc. You can then use relative file paths in your Quarto document (e.g., "/images/myimage.jpg").

If you don't end up needing this directory, be sure to delete/remove it.
